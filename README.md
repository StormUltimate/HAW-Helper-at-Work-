Простой персональный таск‑менеджер:
- задачи: создание / просмотр / редактирование / удаление
- подзадачи с комментариями, статусом, сроком и локальным приоритетом
- приоритизация по методу Эйзенхауэра (квадранты Q1–Q4)
- экспорт / импорт задач и подзадач в JSON

Используется **метод Эйзенхауэра** в самом минимальном виде:
у каждой задачи и подзадачи есть квадрант (Q1–Q4), статус и,
при необходимости, срок. Это не “комбайн”, а максимально
простой инструмент, который помогает быстро разложить задачи
по важности и срочности.

## Установка после клонирования репозитория

Репозиторий:

```text
https://github.com/StormUltimate/HAW-Helper-at-Work-.git
```

Клонирование и первый запуск (Windows / PowerShell):

```powershell
git clone https://github.com/StormUltimate/HAW-Helper-at-Work-.git
cd HAW-Helper-at-Work-
setup_env.bat
```

Скрипт `setup_env.bat` автоматически:
1. создаст виртуальное окружение `.venv`,
2. установит зависимости из `requirements.txt`,
3. предложит сразу запустить сервер разработки.

Если запускали скрипт и выбрали не стартовать сервер сразу,
поднять приложение можно вручную:
 

``` через start run.bat 
или powershell
.\.venv\Scripts\Activate.ps1
py .\app.py
```

Открыть в браузере: `http://127.0.0.1:5000`

## База данных

По умолчанию используется SQLite файл:
`instance\tasks.sqlite3`

Путь к базе можно переопределить переменной среды `TASK_DB_PATH`.

## Импорт задач из JSON

На главной странице есть:
- кнопка **“Экспорт JSON”** — выгружает текущие задачи и подзадачи;
- кнопка **“Импорт JSON”** в шапке — открывает модальное окно
  для вставки JSON.

Можно:
- экспортировать текущие задачи,
- отредактировать JSON вручную или сгенерировать его другим
  инструментом (GPTchat, Claude, DeepSeek и др.),
- вставить фрагмент JSON в модальное окно импорта.

Формат импорта соответствует экспорту. Пример:

```json
[
  {
    "title": "Стройка дома",
    "description": "Кирпич, закупка метериала",
    "quadrant": 2,
    "status": "planned",
    "subtasks": [
      {
        "title": "Купить гвозди 100",
        "quadrant": 1,
        "status": "in_progress"
      }
    ]
  }
]
```

Такой кусок JSON можно вставить в окно импорта: приложение
создаст соответствующие задачи и подзадачи в своём формате.
Это удобно, если планируете задачи где‑то ещё и хотите быстро
перебросить их сюда, не вводя вручную.
